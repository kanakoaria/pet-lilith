# Codex Pet — Lilith

A fan-made, Codex-compatible animated desktop pet inspired by Lilith from *Fate/Grand Order*.

## Install

Copy the `lilith-compatible` folder into your Codex pets directory:

```text
C:\Users\<your-user>\.codex\pets\
```

Restart Codex if the pet is not shown immediately.

## Contents

- `lilith-compatible/pet.json` — pet manifest
- `lilith-compatible/spritesheet.webp` — 9-row animation sheet
- `assets/contact-sheet.png` — animation preview
